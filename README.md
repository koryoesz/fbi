## Installation

Steps to install and set up the project:

1. Run composer:
   ```bash
   Composer install
2. Setup Application environment variables for database:
3. Run migrations
   ```bash
   php artisan migrate
4. Run seeders
   ```bash
   php artisan db:seed
5. Run application
    ```bash
    php artisan serve
6. check swagger documentation
   http://localhost:{port}/api/documentation
